# Dungeons-Dragons-Advisor
An "friendly" advisor to choose what race you should play in DND. Written in CLIPS declarative language

AUTORZY:
Kamil Smolak, indeks: 160093
Adam Zawadzki, indeks: 148102

ŚRODOWISKO URUCHOMIENIOWE:
1. Język/Kompilator: Python 3.14
2. Biblioteki zewnętrzne: clipspy, tkinker

OPIS PROJEKTU:
Aplikacja jest systemem eksperckim pomagającym wybrać rasę w grze Dungeons & Dragons. 
Wykorzystuje silnik wnioskowania CLIPS (plik knowledge.clp) oraz zewnętrzną bazę tekstów (plik data.json). 
Interfejs graficzny został wykonany w bibliotece Tkinter.
